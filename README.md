# 📊 Customer Shopping Behavior – Data Analytics Project

## 📌 Overview

This project demonstrates an **end-to-end data analytics workflow**, covering data loading, exploratory data analysis (EDA), data cleaning, SQL querying, dashboard development, and business reporting.
The goal is to extract meaningful insights from customer shopping behavior data and present them through **interactive dashboards and professional reports**.

---

## 📂 Dataset

* The dataset contains customer shopping and transaction-related information.
* Data was initially loaded and analyzed using **Python**.
* After cleaning, the structured data was stored and queried using **MySQL** for deeper analysis.
---

## 🛠 Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn) – Data loading, EDA, and cleaning
* **MySQL** – SQL queries and data analysis
* **Power BI** – Interactive dashboard and visualizations
* **Jupyter Notebook** – Python analysis
* **PowerPoint & PDF** – Final presentation and report

---

## ⚙️ Project Workflow

1. Load dataset using Python
2. Perform Exploratory Data Analysis (EDA)
3. Clean and preprocess data
4. Run analytical SQL queries in MySQL
5. Build interactive Power BI dashboard
6. Create a detailed report and presentation

---

## ▶️ How to Run the Project

1. **Python Analysis**

   * Open the Jupyter Notebook
   * Run all cells to perform EDA and data cleaning

2. **SQL Analysis**

   * Import the cleaned dataset into MySQL
   * Execute the SQL queries provided in the 'customer_behavior_sql.sql' file

3. **Power BI Dashboard**

   * Open the Power BI file
   * Refresh data connections if required

4. **Report & Presentation**

   * Review the final PDF report- 'Customer_Shopping_Behavior_Report.pdf'
   * Use the 'Customer-Shopping-Behavior-Analysis.pdf' for presentations
